Driver Manager Issue

Important

If you don't see another option in Driver Manager, you'll need to remove the wrong driver and install the correct one manually. Follow the steps below to fix your WiFi connection.

Installation Steps

Follow these 4 simple steps to fix your WiFi

1

Download the Required Packages

On the other computer with internet access, download these .deb packages from the official Ubuntu mirrors:

üì¶ firmware-b43-installer
üì¶ b43-fwcutter

Where to download: These packages are available at packages.ubuntu.com

Pick your Ubuntu version that matches Linux Mint, usually jammy/22.04

2

Transfer Files to USB

Copy both .deb files onto a USB stick. Make sure you can easily locate them later.

Pro Tip: Create a folder on the USB stick named "wifi-drivers" to keep things organized.

3

Install the Packages

On the Linux Mint computer, plug in the USB, open Terminal, and install the packages with these commands:

Terminal Commands 
sudo dpkg -i /path/to/b43-fwcutter*.deb
sudo dpkg -i /path/to/firmware-b43-installer*.deb

Important: Replace /path/to/ with the actual path to your USB stick. It's usually something like /media/your-username/USB-NAME/

4

Reboot Your Computer

Restart your Linux Mint computer to apply the changes.

Terminal Command 
sudo reboot

Success!

After reboot, the WiFi networks should show up in the Network Manager. You should now be able to connect to your WiFi network normally.

Still Having Issues?

Check Your Driver

Open Driver Manager and verify that the correct driver is active.

Verify Installation

Run lsmod | grep b43 in Terminal to check if the driver loaded correctly.

Check System Logs

Use dmesg | grep b43 to see if there are any error messages.
